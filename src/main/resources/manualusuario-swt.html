Introducción

Bienvenido a SoftwareTeca

SoftwareTeca es una solución amplia que permite a los museos de informática catalogar diferentes tipos de piezas de software. El objetivo principal es el ingreso y la modificación de piezas de software como así también proveer informes sobre inventario e información individual de cada pieza. Es un programa que guarda información sobre medios de almacenamiento (cds, disquettes, y cualquier otro formato que considere necesario), software, copias que se hayan creado del software, y sus ubicaciones en el depósito.

A través de un módulo de login, se podrá realizar las altas, bajas y modificaciones de los usuarios del sistema. Sólo aquellos autorizados para catalogar serán los que tendrán acceso a las opciones para agregar y editar objetos. Cualquier otro usuario podrá realizar consultas y exportar informes. El catalogador podrá ingresar software individualmente o de manera masiva cargando un archivo. El inventario generado a partir de estos ingresos es guardado en una base de datos, la cual puede ser consultada para obtener información sobre software y los medios que lo contienen, así como también las copias. Toda la información es almacenada en una base de datos.

Perfil Catalogador

Software

Aclaración: Por software se hace referencia a los programas digitales, no al medio físico. En un medio puede existir uno o más software. El software puede traer añadido drivers o utilidades, como por ejemplo: un driver de una impresora, una actualización o una utilidad opcional que no trae incluida en la pieza de software principal.

Ingresar Software

Para realizar el ingreso de una pieza de software, se debe acceder por el menú Archivo -> Nuevo -> Software. Esto permite ver una pantalla donde se ingresará toda la información del mismo.

Una vez seleccionado el menú “Nuevo Software”, SoftwareTeca nos mostrará la siguiente pantalla de ingreso:

En una primera instancia podremos ingresar los datos propios del software, como ser nombre, versión y los sistemas operativos que son compatibles con ese software. Por otra parte, se pueden completar aquellos datos de los drivers o utilidades que tiene asociado el software. Cabe destacar que esta última sección no es obligatoria ya que puede existir software sin driver o utilidades.

Una vez finalizado el ingreso, presione el botón “Crear Software” para finalizar la carga exitosa del software.

Modificación de Software

Si se requiere modificar un software, desde el inventario de software, presione botón derecho “Modificar Software” y se desplegará la siguiente pantalla:

Tendrá la posibilidad de modificar cualquier dato ingresado en el software e incluso añadir drivers/utilidades. Una vez finalizada la edición puede confirmar los datos ingresados con el botón “Modificar Software”.

Baja de Software

Si desea eliminar algún software, debe ingresar al inventario de software y presionar botón derecho sobre el elemento a eliminar. Luego,seleccione la opción “Eliminar” y confirme su operación.

Volver

Drivers y Utilidades

A un software se le puede agregar y quitar utilidades, drivers y otras piezas que forman parte del mismo pero no son principales.

Existen 3 operaciones válidas sobre los drivers/utilidades

 

Agregar: Se realiza el ingreso de un nuevo driver/utilidad.

Quitar: Se remueve del software aquel driver/utilidad seleccionado.

Eliminar todos: Remueve todos los drivers/utilidades asociados a ese software.

Volver

Medios

Alta de Medios

Para ingresar un nuevo medio, debe ir al menú “Archivo -> Nuevo -> Medio”

Para ingresar un medio debe completar 3 solapas. La primera consta básicamente de las características del medio como ser, una imágen, código, el nombre y si el medio es original, mixto (algunos elementos del medio son originales) u otros. La primera vez que se hace click en la imágen, el sistema pide que se seleccione la carpeta donde desea guardar las imágenes. Luego de seleccionarla, puede elegir la imágen que desea aplicar.

En la segunda sección se ingresan los detalles físicos del medio. En este módulo se encontrarán aquellos datos relacionados con el formato del medio, la cantidad de partes que tiene, su ubicación en el museo y las observaciones que el usuario considere pertinentes.

Por último, se seleccionan aquellos software que contiene ese medio. Una vez finalizado el ingreso de datos, presione el botón “Ingresar Medio” para concluir exitosamente la carga del medio.

 

Volver

Modificación de Medios

Si desea modificar algún dato, desde el inventario de medios, seleccione el medio que desea editar y presione la opción “Modificar Medio”. En este caso, se habilitarán las 3 secciones donde el usuario podrá editar el dato que desee.

                 


Luego de finalizar la edición, presione el botón “Actualizar Medio” para confirmar los cambios.

Volver

Baja de Medios

Desde el inventario de medios, seleccione el medio, presione botón derecho y elija la opción “Eliminar”. Luego, confirme la operación para eliminar definitivamente el medio.

Cambiar imagen default        

Si quiere cambiar la imagen que se muestra por default cuando un medio no tiene imagen, debe entrar a la carpeta seleccionada para el almacenamiento de imágenes y guardarla bajo el nombre “no-image-available.png”.

Volver

Copias

Cada medio puede tener una o varias copias. Para gestionar las copias de los medios, se debe apretar botón derecho sobre el medio y se desplegará la opción “Administrar Copias”. A continuación, se detallan las opciones que se pueden realizar:

Alta de Copias

Para crear una copia, se debe completar el formato y la ubicación que tendrá este nuevo elemento. Adicionalmente, se puede incorporar una breve descripción para detallar algún aspecto de esta copia. Una vez finalizado el ingreso, confirme la operación presionando el botón “Crear Copia”.

Volver

Modificación de Copias

Si desea modificar los datos de la copia, puede modificar el formato o la ubicación mediante la selección de otras opciones en el combobox o bien agregar una nueva descripción a la copia. Para confirmar los datos editados, presione el botón “Actualizar Copia” y se guardan las modificaciones realizadas.

Baja de Copias

Para eliminar una copia existente, debe seleccionar la copia a remover y luego confirmar la operatoria presionando el botón “Eliminar Copia”.

Volver

Ver copias de medio

En este módulo, únicamente podrá visualizar las copias que tiene el medio seleccionado. No podrá realizar ninguna edición de las mismas.

Volver

Perfil Catalogador - Normalización

Este perfil permite, además de actualizar la información contenida sobre las piezas, modificar, agregar o eliminar elementos más sensibles de normalización del contenido. El usuario asignado con este perfil podrá ingresar nombres de sistemas operativos, formatos físicos de medios y ubicaciones.

Ubicación

Para poder ingresar a la edición de Ubicaciones, seleccione el menú “Editar” y luego “Ubicaciones”

Alta Ubicación

Para ingresar una nueva ubicación al sistema, simplemente se debe ingresar el código de esa nueva ubicación y una breve descripción que detalle alguna característica de ese sitio. Luego presione la opción agregar para confirmar la operación.

Baja de Ubicación

Para dar de baja la ubicación, seleccione el elemento a remover y presione el boton “Quitar”.

Tenga en cuenta que las ubicaciones se encuentran asociadas a los medios por lo que el sistema le comunicará si está eliminando alguna ubicación asociada a un medio.


Volver

Formatos

Para ingresar a la administración de formatos, seleccione el menú “Editar” opción “Formatos”

Alta de Formatos

Para ingresar un nuevo formato, debe completar su nombre y luego presionar el botón Agregar para confirmar la operación.

Baja de Formatos

Si desea eliminar un formato, seleccione la opción que desea remover y luego presione el botón “Quitar”.

Luego se le solicitará que reemplace el formato que está eliminando en aquellos medios que están almacenados con este formato. Ingrese el nuevo valor del formato. En caso de no poseer ningún formato que lo reemplace, deje el espacio en blanco.

Volver

Sistema Operativo

Para ingresar a la administración de los sistemas operativos, ingrese al menú “Editar”

Alta de Sistema Operativo

Para dar de alta un nuevo sistema operativo, simplemente se debe ingresar el nombre y presionar el botón “Agregar”.

Baja de Sistema Operativo

Si desea eliminar un sistema operativo, debe seleccionar el elemento a remover y luego presionar el botón “Quitar”.

Cabe destacar que al eliminar un sistema operativo, el sistema le solicitará que reemplace este sistema operativo por otro vigente. En caso de no querer reemplazarlo, deje el espacio en blanco.

Volver

Perfil Administrador

Usuarios

Para ingresar al módulo de Administración de usuarios, debe seleccionar el menú “Archivo” y luego “Opciones - Administración de Usuarios”

Alta de Usuarios

Para dar de alta un usuario en el sistema, debe ingresar el nombre de la persona, el nombre de usuario y una contraseña asociada a esa cuenta.

Adicionalmente, se solicitará que le asigne un perfil a esa nueva cuenta. El detalle de los perfiles se describe a continuación:

Consultivo: 

Sólo se le permite ver y exportar información de las piezas. No puede modificar o eliminar.

Catalogador:

Permite ingresar, modificar y eliminar medios y software. No permite agregar datos sensibles a la normalización.

Catalogador - Normalización: 

Permite ingresar, modificar y eliminar tanto medios y software como formatos, ubicaciones y sistemas operativos (más riesgoso).

Administrador:

Permite la administración de los usuarios.  

Una vez finalizado el ingreso, presione el botón “Crear Usuario” para confirmar la operación.

Modificación de Usuarios

Ingresando al menú “Modificar Usuario” tendrá la posibilidad de editar el tipo de perfil que posee el usuario. Una vez realizadas las modificaciones necesarias, presione el botón “Modificar Usuario” para confirmar la operación.

Baja de Usuarios

Para realizar la baja de un usuario, se debe seleccionar la cuenta en cuestión y presionar el botón “Eliminar”.

Debido a que es una operación importante, el sistema le solicitará las credenciales de administración nuevamente para eliminar definitivamente el usuario. Ingrese las credenciales y presione el botón “Login” para confirmar la operación.
